    Улучшение разметки

    Добавление выравнивания, надстрочного/подстрочного скрипта, сноски, списка задач, текста, блок-схемы, диаграммы, выделения и поддержка презентации в Markdown

    Слайд-страница

    Добавление страниц слайдов для отображения того, что вам нравится

    Улучшение макета

    Добавление хлебных крошек, нижнего колонтитула, улучшенной панели навигации, улучшенной навигации по страницам и т. д.

    Просмотры страниц и комментарии

    Запустите статистику просмотров страниц и поддержку комментариев с помощью Waline

    Информация о статье

    Добавление автора, даты написания, время чтения, количество слов и другой информации в свою статью

    Поддержка блога

    Список ваших статей с их датами, тегами и категориями с некоторыми потрясающими макетами

    Пользовательский цвет темы

    Поддержка пользовательских цветов темы и позволяет пользователям переключаться между предустановленными цветами темы

    Темный режим

    Свободно переключайтесь между светлым и темным режимами

    Полная поддержка A11y

    Полная поддержка специальных возможностей на вашем сайте

    Шифрование статьи

    Шифруйте свои статьи на основе ссылок на страницы, чтобы их мог видеть только тот, кому вы хотите

    Поиск

    Поддержка docsearch и клиентского поиска

    Кнопка копирования

    Копируйте коды одним кликом в блоках кодов

    Предпросмотр изображения

    Поддержка просмотра, масштабирования, обмена изображениями на странице, например, в галерее

    СЕО-улучшения

    Оптимизация страниц для поисковых систем

    Карта сайта

    Создание потока, чтобы пользователи могли подписаться на него

    Поддержка PWA

    🛠Установка

    Создайте проект vuepress-theme-hope в каталоге [dir] по текущему пути:

    pnpm
    pnpm create vuepress-theme-hope [dir]
    yarn
    yarn create vuepress-theme-hope [dir]
    npm
    npm init vuepress-theme-hope [dir]

    Чтобы добавить vuepress-theme-hope в качестве конструктора документов в существующий проект, выполните следующую команду в корневом каталоге проекта:

    pnpm
    pnpm create vuepress-theme-hope add [dir]
    yarn
    yarn create vuepress-theme-hope add [dir]
    npm
    npm init vuepress-theme-hope add [dir]

    🚀Использование

    TS
    // .vuepress/config.ts
import { defineUserConfig } from "vuepress";
import { hopeTheme } from "vuepress-theme-hope";

export default defineUserConfig({
  theme: hopeTheme({
    // your theme config here
  }),
});
    JS
    // .vuepress/config.js
import { hopeTheme } from "vuepress-theme-hope";

export default {
  theme: hopeTheme({
    // your theme config here
  }),
};
